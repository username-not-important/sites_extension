# AI Sites Enhancer

A small custom Chrome/Edge extension built out of need to improve the visual and functional experience of some AI websites used in Iran.

The extension currently targets:

- AvalAI
- GapGPT

## Features

- Cleaner layout and spacing for supported AI chat pages
- Visual tweaks for better readability
- GapGPT message navigation controls
- Better code block handling, including collapse/expand controls
- Extra copy controls for code blocks and inline code
- Site-specific styling without affecting unrelated websites

## Installation

This extension is not published on the Chrome Web Store. Install it manually in developer mode.

### Chrome

1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on **Developer mode** in the top-right corner.
4. Click **Load unpacked**.
5. Select this repository folder.
6. Open AvalAI Chat or GapGPT and refresh the page.

### Microsoft Edge

1. Download or clone this repository.
2. Open Edge and go to `edge://extensions`.
3. Turn on **Developer mode**.
4. Click **Load unpacked**.
5. Select this repository folder.
6. Open AvalAI Chat or GapGPT and refresh the page.

## Updating

After changing files or pulling new updates:

1. Go back to the browser extensions page.
2. Find **AI Sites Enhancer**.
3. Click the reload button.
4. Refresh the target website tab.

## Notes

- This is a personal utility extension, made for practical daily use.
- It only runs on the websites listed in `manifest.json`.
- No build step is required.
